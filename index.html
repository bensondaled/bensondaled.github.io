Ben Deverett @ GitHub

This page is a hub for some of my projects.

  • EEG Instrument (2011): final project for MUMT306 at McGill. A MIDI musical instrument controlled by signals from a single-electrode EEG routed through Python.

  • Tempo Detection (2012): final project for MUMT307 at McGill. A Matlab implementation of a tempo-detection algorithm.

  • pyfluo (2013-ongoing): a Python library for analysis of 2-photon calcium imaging data.

  • trigger-interface (2014-ongoing): a small Python module for interfacing with DAQ and cameras during microscopy + conditioning experiments.

  • three-chamber (2014-ongoing): A set of python scripts and tools for acquiring and analyzing mouse 3-chamber social experiments.

  • imaging-simulation (2014-ongoing): an increasingly comprehensive modeling effort of granule cell 2-photon calcium imaging.
